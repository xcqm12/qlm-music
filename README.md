# 七零喵聚合音源 · 超级整合版

## 📖 简介

七零喵聚合音源是一个为洛雪音乐（LX Music）设计的超级整合版音源脚本，集成了多种音源 API，提供多源回退、智能缓存、相似歌曲搜索等功能，确保音乐播放的稳定性和可用性。

### ✨ 主要特性

- **多音源聚合**：整合 ikun、聚合 API、qorg、星海、溯音、六音、独家音源、长青 SVIP、念心 SVIP、野花野草、Meting、汽水 VIP、肥猫系列、梓澄系列、無名、小熊猫、Free listen 等 20+ 音源
- **网易云盘支持**：支持网易云音乐云盘歌曲的搜索、播放和歌词获取（v7.0.3+）
- **智能回退**：当一个音源失败时自动切换到下一个可用音源
- **智能缓存**：LRU 缓存机制，减少重复请求，提升响应速度
- **相似歌曲搜索**：当主链失败时自动搜索相似歌曲
- **多平台支持**：支持 QQ 音乐、网易云音乐、酷我音乐、酷狗音乐、咪咕音乐、汽水 VIP 等主流平台

## 📥 安装使用

### 前置要求

- [洛雪音乐助手](https://github.com/lyswhut/lx-music-desktop)（桌面版）或 [洛雪音乐移动版](https://github.com/lyswhut/lx-music-mobile)
- 洛雪音乐版本建议 v2.0.0 及以上

### 安装步骤

1. 打开洛雪音乐助手
2. 进入「设置」→「基本设置」
3. 找到「音乐来源」或「音源设置」
4. 选择「自定义源」或「导入源」
5. 将本脚本的原始链接或本地文件路径填入

**推荐使用原始链接（自动更新）：**
```
https://raw.githubusercontent.com/xcqm12/qlm-music/main/qlm-v7.0.4.js
```

> 💡 如果没有 GitHub 访问条件，可以下载本仓库的 `qlm-v7.0.4.js` 文件，选择「本地文件」方式导入。

### 版本选择

| 版本 | 说明 | 推荐场景 |
|------|------|----------|
| v7.0.4 | 最新完整版，支持网易云盘，优化 Cookie 管理 | 推荐使用 |
| v7.0.3 | 新增网易云盘支持 | 需要使用云盘功能 |
| v7.0.2 | 优化音源逻辑，修复部分问题 | 稳定版本 |
| v7.0.1 | 修复 musicUrl 支持 | 兼容性较好 |
| v5.0.2 | 早期聚合版本 | 基础功能 |

### 网易云盘配置（v7.0.3+）

如需使用网易云盘功能，需要配置网易云音乐的 Cookie：

1. 登录网页版网易云音乐（https://music.163.com）
2. 打开浏览器开发者工具（F12）
3. 切换到「Network」（网络）标签
4. 刷新页面，找到任意请求
5. 在请求头中找到 `Cookie` 字段，复制完整内容
6. 在洛雪音乐中通过自定义源提供的接口设置 Cookie

> ⚠️ Cookie 包含个人账号信息，请勿分享给他人。Cookie 有效期通常为 24 小时，过期后需重新获取。

## 🔧 常见问题

### Q1：部分歌曲无法播放？

A：本音源采用多源回退机制，会自动尝试所有可用音源。如果仍然失败，可能是所有音源均不支持该歌曲，建议尝试切换其他平台搜索。

### Q2：网易云盘无法使用？

A：请检查是否正确配置了网易云音乐 Cookie，且 Cookie 未过期。Cookie 有效期通常为 24 小时，过期后需要重新获取。

### Q3：播放卡顿或加载慢？

A：可尝试在设置中降低默认音质（如从 flac 降至 320k），或检查网络环境。

### Q4：提示「所有音源均失败」？

A：可能是网络问题或音源服务临时不可用，建议稍后重试。如持续出现，请检查洛雪音乐版本是否为最新。

### Q5：导入后没有反应？

A：请检查洛雪音乐版本是否支持自定义音源，并确保导入的链接或文件路径正确。
